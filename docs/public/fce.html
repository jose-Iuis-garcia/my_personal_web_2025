Course description

Fundamentals of Computer Engineering is a first-year subject within the Double Degree in Mathematical Engineering and Computer Engineering at Universidad Francisco de Vitoria. The course provides a broad introduction to how computer systems operate, combining technical knowledge with social and ethical perspectives.

Unlike purely technical subjects, this course integrates theoretical concepts with practical skills that can be directly applied. Students are introduced to real-world tools and workflows through a final project known as Practical Work II, where technologies such as Git, HTML, and CSS are used.

Course structure

Part I – Theoretical foundations

Block 1: Computer engineering in society

  • Historical background and future outlook of computer engineering
  • Basic concepts and core definitions
  • Evolution of information and communication technologies
  • Professional skills required for computer engineers
  • Ethical principles and professional responsibility

Block 2: Systems and application paradigms

  • Software development beyond coding
  • Structure of computer systems: hardware, software, operating systems, and networks
  • Current paradigms and emerging technological trends

Block 3: Information and data

  • Digital representation of information
  • Numerical systems and data encoding
  • Introduction to Big Data concepts
  • Legal and ethical aspects of data management

Part II – Practical skills

Block 1: Software development basics

  • Introduction to web development
  • Version control systems: Git and GitHub

Block 2: Web development with HTML5

  • HTML5 fundamentals
  • Basic HTTP concepts and tags
  • Content organization and semantic structure

Block 3: Design and layout with CSS3

  • CSS3 core concepts
  • Selectors and styling rules
  • Box model and element positioning

Practical Work II

Application of acquired knowledge through the development of a functional website using HTML and CSS.
